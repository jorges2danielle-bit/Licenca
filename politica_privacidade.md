# Política de Privacidade - Paganini Orchestra

A presente política descreve como tratamos os dados coletados para ativação da licença do software.

1. **Coleta de Informações:** Coletamos única e exclusivamente o número de identificação da conta do MetaTrader 5 (login) fornecido pelo cliente para fins de validação e ativação da licença de uso do software. Não temos acesso a senhas, chaves privadas ou saldos.
2. **Uso dos Dados:** O ID da conta MT5 é utilizado apenas pelo nosso servidor de licenças para autorizar ou bloquear a execução do executável local.
3. **Compartilhamento:** Não compartilhamos, vendemos ou divulgamos quaisquer dados de clientes com terceiros.
4. **Segurança:** Adotamos medidas de criptografia no executável para garantir que as informações de licença permaneçam seguras.
