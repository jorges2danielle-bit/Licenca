Paganini Orchestra - Central de Documentação e Licenciamento
Este repositório contém a documentação legal e a central de validação de licenças para a suite de trading quantitativo Paganini Orchestra (que inclui os robôs Paganini e XAUUSD, além do dashboard de telemetria ECC).

📄 Documentos Legais
Para a transparência e conformidade com os nossos processadores de pagamento e usuários, os seguintes documentos oficiais estão disponíveis neste repositório:

Termos de Serviço
 - Regras, diretrizes e responsabilidades ao utilizar nossos softwares e serviços de trading.
Política de Privacidade
 - Como lidamos com a segurança dos dados e informações dos nossos usuários.
🔒 Sistema de Licenciamento (Interno)
Este repositório também hospeda de forma segura os dados para a central de validação de licenças dos robôs da suite Paganini Orchestra. A validação de contas é realizada de forma automática e integrada para garantir que apenas usuários ativos tenham acesso aos sistemas.

🛡️ Proteção de Garantia (Trial de 7 Dias)
Para proteção contra abusos e fraudes de reembolso, o sistema opera sob as seguintes diretrizes:

Primeiros 7 dias (Período de Garantia): O robô opera exclusivamente em contas do tipo DEMO (Demonstração).
A partir do 8º dia: A licença é atualizada para o tipo REAL, liberando a operação com capital real na plataforma MetaTrader 5.
Se um usuário tentar operar uma licença do tipo Demo em uma conta Real antes do prazo regulamentar, o software bloqueará a execução automaticamente para proteção mútua.
Para suporte ou dúvidas comerciais, utilize os canais oficiais de atendimento.
